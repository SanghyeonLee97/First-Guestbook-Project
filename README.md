## 축구 팬들을 위한 게시판
#### 기간: 2023.11. ~ 2023.12.

### 개요
 - 교육에서 배운 MVC2 모델을 적용하여 자바를 기반으로 게시판 구현
 - 축구 팬들을 위한 소통의 공간 제공
